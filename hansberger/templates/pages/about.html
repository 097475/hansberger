About

HansBerger is a local web application built with the Python web framework Django. HansBerger implements an automatized pipeline which simplifies Topological Data Analysis (TDA) by providing an easy to use interface for various operations data analysis and data visualization.

The application functionalities are:

  • Visualization of text, CSV and EDF datasets

  • Upload of precomputed distance matrices

  • Vietoris-Rips Filtration with Ripser

  • Clique Weight Rank Filtration with Ripser

  • MAPPER algorithm with Kepler Mapper

  • Computation of entropy values of a persistence diagram

  • Computation of bottleneck distance between persistence diagrams with Persim

  • Saving of results to a relational database

The name of the application, HansBerger, is a reference to the homonym german psychiatrist, known for having recorded the first brain waves and for the invention of electroencephalography. Berger worked in many fields of research: neurology, brain circulation, psychology. His main contribute to medicine was the research on electrical activity in the brain and the discovery of EEG. The first recording of human brain waves, performed by Berger, was carried out using an instrument which he called "electroencephalograph", producing a graphic representation called "electroencephalogram".
